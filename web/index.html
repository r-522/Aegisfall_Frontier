Aegisfall Frontier

侵攻型タワーディフェンス RPG。Godot 4製。開発中。

数百年前、人類は魔王軍に敗北した。生き残った人々は巨大結界都市に身を寄せている。 プレイヤーは大陸 Eldrath を奪還する Frontier Legion の一員として、前線拠点を築き、 タワーを並べ、徐々に魔王城へ攻め込む。

従来のTDと違って防衛地点が移動する。プレイヤー自身も戦う。

遊ぶ

▶ ブラウザで起動 (シングルプレイのみ)

注意: 初回読み込みに30秒〜1分かかる (約43MBのwasm)。 マルチプレイは ENet/UDP のためブラウザでは無効。デスクトップ版が必要。 動作確認: Chrome / Edge / Firefox。
