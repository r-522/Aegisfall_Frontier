世界観

数百年前、人類は魔王軍に敗北した。生き残った人々は巨大結界都市に身を寄せ、 失われた大陸の奪還を諦めずに「Frontier Legion」を編成した。

プレイヤーはその一員として、前線拠点を築き、補給線を維持し、 タワーを展開しながら、徐々に魔王城へ攻め込む。 従来のTDと異なり、防衛地点は移動し、プレイヤー自身が戦闘主体となる。

特徴

39 プレイヤークラス

近接 / 魔法 / 支援 / 敏捷 / 特殊 の5カテゴリ。Fighter から Trickster まで、各クラス固有のスキル4種を搭載。

15 タワー × 4カテゴリ

防壁 / 攻撃塔 / 支援塔 / 特殊塔。建築フェーズで陣形を組み、Wave 防衛フェーズで耐える。

協力型オンライン

ENet サーバー権威モデル。30Hz tick / 60fps クライアント補間で同期的アクション。

ハック&スラッシュ

6段階レアリティ / 7属性 / 13カテゴリのアイテム + 17種類のアフィックス。深い装備育成。

ゲームフェーズ

Lobby → Exploration → Build → Wave Defense → Counter Attack → Results の6段階ループ。

Diablo IV 風 UI

半透明黒 + 金属ゴールドアクセント。ヒットストップ・カメラシェイク・部位リアクション完備。
